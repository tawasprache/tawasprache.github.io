Tawa

Tawa ist eine einfache funktionale Programmiersprache. Es sieht wie aus das: 

paket Haupt zeigt alles

typ Farbe ist
    a: Vielleicht[Ganz]

    | RGB(r: Ganz, g: Ganz, b: Ganz)
    | HSV(h: Ganz, s: Ganz, v: Ganz)
beende

funk ZuRGB(f: Farbe)
    passt f zu
        #RGB(r, g, b) =>
            f

        #HSV(h, s, v) =>
            #RGB(h, s, v) { a = f.a }
    beende

funk Haupt()
    Debuggen:druck(
        Liste:map([1, 2, 3], \(x: Ganz): Ganz => x + 1)
    )
