Tawa / Lernen

Werten

Werten sind die Bausteine von Tawa. Das beinhaltet Dinge wie `42`, `#Wahr`, und `”Hallo!”`.

Ganze Zahlen

Tawa versteht nur ganze Zahlen. Man kann normale Arithmetik mit Tawa tun. 
>>> 1 + 1
2

>>> 1 + 1 * 2
3

>>> (1 + 1) * 2
4

Zeichenkette

Text ist durch Zeichenketten dargestellt. Man kann zwei Zeichenketten verketten mit ++. 
>>> "hallo"
"hallo"

>>> "hallo" ++ "welt!"
"hallowelt!"

Funktionen

Funktionen erlaubt man, Werten zu ändern. 
funk Grüßen(name: Zeichenkette): Zeichenkette
    "Hallo, " ++ name ++ "!"
>>> Grüßen("Jan")
Hallo, Jan!

Typen

Typen erlaubt man, Werten zu gruppen. Sie sieht wie aus das: 
typ Mensch ist
    name: Zeichenkette
    alter: Ganz
beende
>>> #Mensch { name = "Jan", alter = 18 }
#Mensch { name = "Jan", alter = 18 }
